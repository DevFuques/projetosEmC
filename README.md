## ⚙️ Pré-requisitos

Antes de executar os códigos, verifique se você possui um compilador C instalado.

### 🔍 Verificando o GCC

No terminal:
```bash
gcc --version
```

Se aparecer algo semelhante a:
```bash
gcc (Ubuntu 13.2.0) 13.2.0
```

✅ Tudo pronto para continuar.

---

## 🐧 Instalando GCC

### Ubuntu/Debian
```bash
sudo apt update
sudo apt install build-essential -y
```

### Fedora
```bash
sudo dnf install gcc gcc-c++ make
```

### Windows
Instale o MinGW:
👉 https://www.mingw-w64.org/

Ou utilize:
- Visual Studio Community
- WSL (Windows Subsystem for Linux)

## ⚙️ Como executar os códigos

### 1️⃣ Clonar o repositório
```bash
git clone https://github.com/DevFuques/projetosEmC.git
```

```bash
cd projetosEmC
```

### 2️⃣ Compilar um arquivo
Exemplo:
```bash
gcc tabuada.c -o tabuada
```

### 3️⃣ Executar

#### Linux/macOS
```bash
./tabuada
```

#### Windows
```bash
tabuada.exe
```
